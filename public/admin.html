Webhook

Configure o recebimento de pagamentos

URL do Webhook

http://localhost:3462/webhook
  • Cole esta URL no painel da PushinPay em Configurações → Webhook
  • Em produção substitua localhost:3462 pelo seu domínio
  • Use ngrok para testes locais: ngrok http 3462

Eventos suportados

  • transaction.paid — Pagamento confirmado → envia produto por email automaticamente
  • transaction.expired — Cobrança expirada
  • transaction.refunded — Estorno realizado

Configurar Email (SMTP)

Edite o arquivo .env com suas credenciais:

SMTP_HOST=smtp.gmail.com
SMTP_PORT=587
SMTP_USER=seuemail@gmail.com
SMTP_PASS=sua_senha_app   ← Senha de app do Google
EMAIL_FROM=seuemail@gmail.com
  • Gmail: ative verificação 2 etapas → gere Senha de App
  • Variáveis ficam no .env — nunca comite esse arquivo
